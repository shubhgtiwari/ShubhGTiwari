## üìä Featured Projects

### [Protein Family Classification](https://github.com/ShubhGTiwari/Protein_Classification)
Machine learning pipeline for predicting protein families from amino acid sequences using Random Forest, SVM, and XGBoost. Includes feature engineering, model evaluation, and ROC analysis.

### [Artemis Lunar Sample Analysis](https://github.com/ShubhGTiwari/artemis-lunar-analysis)
NASA Artemis mission planning ‚Äî analyzed 383 kg of Apollo lunar samples to generate data-driven collection recommendations for future Moon missions.

### [Universe Database](https://github.com/shubhgtiwari/Celestial_Bodies-Database)
Relational database modeling the cosmos. PostgreSQL schema with foreign key constraints capturing the hierarchy: Constellation ‚Üí Galaxy ‚Üí Star ‚Üí Planet ‚Üí Moon.

### [India Copper Trade Analytics](https://github.com/shubhgtiwari/Export-and-Import-of-Copper)
International trade flow analysis using UN Comtrade API. Revealed India's $1.5B trade deficit and mapped critical trading partners.

### [KPMG Customer Segmentation](https://github.com/shubhgtiwari/KPMG_Virtual_Internship)
RFM analysis segmenting 20,000+ customers into actionable groups. Built Power BI dashboard for stakeholder decision-making.
